Box A

Here, the .wrapper-a class is set to justify-items:end, which will resize each div to fit its own contents, and also push them to the right within each column. Columns are set to 1fr 2fr 1fr, so that Box B is pushed to the right in the 2nd column.

Justify-items vs. Justify-content
justify-items: resizes grid areas each to itself, smaller for less content;
justify-content: resizes grid areas to the parent, same size for all content.
